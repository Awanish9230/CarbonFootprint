# EcoReduce - Carbon Footprint Tracker 🌍

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A full-stack MERN (MongoDB, Express.js, React, Node.js) application designed to help individuals track, understand, and reduce their personal carbon emissions. The application provides a user-friendly interface to log daily activities, visualize emission trends, participate in community challenges, and discover sustainable alternatives. 🌿

## ✨ Features

*   **Carbon Emission Tracking:** Log daily activities related to transportation, energy consumption, shopping, and diet to calculate your carbon footprint. 📊
*   **Emission Visualization:** Visualize your emission trends over time with interactive charts and summaries. 📈
*   **Personalized Recommendations:** Get AI-powered and rule-based suggestions on how to reduce your carbon footprint based on your activity logs. 💡
*   **Community Engagement:** Share your progress, tips, and ideas with other users in the community forum. 🗣️
*   **Gamification:** Earn points, badges, and rewards for completing challenges and achieving milestones. 🏆
*   **Virtual Eco Garden:** Plant trees and level up your virtual garden as you reduce your carbon emissions. 🌳
*   **Marketplace:** Redeem your earned points for eco-friendly products and rewards. 🎁
*   **Greener Alternatives:** Discover eco-friendly alternatives for various items and activities. ♻️
*   **Leaderboard:** Compete with other users on national and state leaderboards. 🏅
*   **Dark Mode:** Enjoy a visually comfortable experience with a dark mode option. 🌙
*   **Responsive UI:** Access the application seamlessly on various devices with a responsive design.📱

## 🚀 Installation

### Prerequisites

*   Node.js (version 18 or higher) ⚙️
*   npm (Node Package Manager) 📦
*   MongoDB (running locally or a connection string) 🍃

### Backend Setup

1.  Clone the repository:
    ```bash
    git clone https://github.com/Awanish9230/CarbonFootprint.git
    cd CarbonFootprint
    ```
2.  Navigate to the backend directory:
    ```bash
    cd backend
    ```
3.  Copy the example environment variables file and configure it:
    ```bash
    cp .env
    ```
    Edit the `.env` file with your MongoDB URI and JWT secret:
    ```
    PORT=5000
    MONGO_URI=mongodb://localhost:27017/carbon_footprint_tracker
    JWT_SECRET=your_long_random_secret
    ```
4.  Install the backend dependencies:
    ```bash
    npm install
    ```
5.  Run the backend development server:
    ```bash
    npm run dev
    ```
6.  Seed the emission factors (optional but recommended):
    ```bash
    node scripts/seedItems.js
    node scripts/seedEmissionFactors.js
    ```

### Frontend Setup

1.  Navigate to the frontend directory:
    ```bash
    cd frontend
    ```
2.  Install the frontend dependencies:
    ```bash
    npm install
    ```
3.  Run the frontend development server:
    ```bash
    npm run dev
    ```
4.  The Vite dev server proxies `/api` to `http://localhost:5000` (configured in `vite.config.js`).

## 💡 Usage

1.  Start the backend server:
    ```bash
    cd backend
    npm run dev
    ```
2.  Start the frontend development server:
    ```bash
    cd frontend
    npm run dev
    ```
3.  Open your browser and go to `http://localhost:5173` to access the application.

*   **Register** a new account or **Login** with an existing one.
*   **Log** your daily activities related to transportation, energy, shopping, and food.
*   **View** your carbon footprint trends and personalized recommendations on the dashboard.
*   **Participate** in community challenges and earn badges.
*   **Redeem** points in the marketplace for eco-friendly rewards.

## 📜 API Reference

The backend provides the following API endpoints:

### Auth Routes

*   `POST /api/auth/register`: Register a new user.
*   `POST /api/auth/login`: Login an existing user.
*   `GET /api/auth/profile`: Get the profile of the logged-in user.
*   `POST /api/auth/upload`: Upload profile image.

### Emission Routes

*   `POST /api/emissions/log`: Log a new emission entry.
*   `GET /api/emissions/summary?range=daily|weekly|monthly|yearly`: Get a summary of emissions for a given range.
*   `GET /api/emissions/search-item?q=...`: Search for emission factors.
*   `GET /api/emissions/recommendations`: Get personalized recommendations to reduce emissions.

### Leaderboard Routes

*   `GET /api/leaderboard/national`: Get the national leaderboard.
*   `GET /api/leaderboard/state/:state`: Get the leaderboard for a specific state.

### Community Routes

*   `GET /api/community/posts`: Get all community posts.
*   `POST /api/community/posts`: Create a new community post.
*   `POST /api/community/posts/:id/like`: Like a community post.
*   `POST /api/community/posts/:id/comments`: Add a comment to a community post.

### Gamification Routes

*   `POST /gamification/log-action`: Log a user action and update points, streak, and garden.
*   `POST /gamification/complete-challenge`: Mark a challenge as completed and award points and badges.
*   `POST /gamification/redeem-reward`: Redeem a reward for a user.
*   `GET  /gamification/user/:id`: Get user data with gamification details.
*   `POST /gamification/generate-daily-challenges`: Generate daily challenges for a user.

### Item Routes

*   `GET /api/items/`: Get all carbon emission items.
*   `GET /api/items/search?q=`: Search carbon emission items by name.
*   `GET /api/items/suggestions?q=`: Get suggestions for items to search.
